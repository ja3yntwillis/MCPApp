## Getting Started

1. **Clone the repository:**
```sh
git clone https://github.com/ja3yntwillis/MCPApp.git
cd MCPApp
```

2. **Restore dependencies:**
```sh
dotnet restore
```

3. **Build the project:**
```sh
dotnet build
```

4. **Run the application:**
```sh
dotnet run
```
